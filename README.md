# Projeto Nexus

Painel interativo que mapeia o potencial profissional e aponta oportunidades.

Instruções

- Instalar dependências:

```powershell
cd 'c:\Users\renan\Projetos\my-app'
npm install
```

- Rodar em modo desenvolvimento:

```powershell
npm run dev
# Acesse http://localhost:5173 ou a porta informada pelo Vite
```

- Build para produção:

```powershell
npm run build
```

Observações

- O projeto usa Vite + React + Tailwind. O dev server pode abrir em outra porta se 5173 já estiver em uso.
- Se você ver um aviso sobre `postcss.config.js` e `MODULE_TYPELESS_PACKAGE_JSON`, adicione `"type": "module"` em `package.json` para suprimir esse aviso (opcional).
